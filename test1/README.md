# Student Management Project

A basic Flutter project to manage student records.

## Features
- **View Students**: See a list of all enrolled students.
- **Add Student**: Add new students with ID, Name, and Age.
- **Edit Student**: Update details of existing students.
- **Delete Student**: Remove students from the list.
- **Search**: Quickly find students by name or ID.

## Project Structure
- `lib/main.dart`: Entry point of the application.
- `lib/Entities/Student.dart`: Data model for Student.
- `lib/student_list_screen.dart`: Main UI and logic for student management.
